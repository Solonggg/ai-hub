# Create Feat

## 触发词

`新建 feat`、`create feat`

## 执行步骤

1. 通过 `date` 命令获取当前日期，计算：
   - **年编号** = 当前年份 - 2022（如 2026 → 4，2027 → 5）
   - **月** = 当前月份（1-12，不补零）

2. 扫描项目根目录 `doc/feat/` 下所有文件名匹配 `feat_v{年编号}.{月}.*.md` 的文件

3. 计算月版本序号：
   - 若无当月文件 → 序号 = 1
   - 若有当月文件 → 取最大序号 + 1

4. 直接创建文件 `doc/feat/feat_v{年编号}.{月}.{序号}.md`，使用下方模板，将 `{版本号}` 替换为 `{年编号}.{月}.{序号}`

5. 告知用户文件已创建及完整路径

## 注意事项

- 直接创建，不需要用户确认版本号
- 如果 `doc/feat/` 目录不存在，先创建目录
- 模板文件：`references/feat-template.md`，读取后将 `{版本号}` 替换为实际版本号
