# 自动分类规则

## 三类业务域关键词词表

每类约 15 词，可按需扩。命中次数加总即词频得分。

### 病案
```
病案、病历、归档、质控、ICD、首页、入院记录、出院记录、病程、病历首页、终末病历、运行病历、电子病历、EMR、归档质控
```

### 护理
```
护理、护士、体征、医嘱执行、护理记录、护理评估、护理计划、入院评估、压疮、跌倒、风险评估、护理交班、护理质控、生命体征、护理单
```

### 健康档案
```
健康档案、居民档案、公卫、慢病、随访、家庭医生、签约、儿童保健、老年人体检、孕产妇、预防接种、基本公卫、电子健康档案、EHR、随访记录
```

## 评分逻辑

```
score(类) = Σ (词在标题中出现次数 × 3) + Σ (词在正文中出现次数)
```

- 标题命中权重 ×3，正文命中权重 ×1
- 所有匹配按词面相等（含中英文混用），区分大小写仅对英文缩写（ICD/EMR/EHR 大写）
- 同一词多次出现按实际次数计入，不去重

## 判定流程

```
1. 计算三类得分 s1, s2, s3
2. 取最高分 max 与次高分 sub
3. 若 max == 0          → 停下提问（无任何命中）
4. 若 max - sub < 2     → 停下提问（命中不显著）
5. 否则 → 取 max 对应分类
```

阈值 2 可调。意图是排除"两类各命中 1 词"这种均匀分布。

## 歧义提问模板

```markdown
该文档无法自动判定归档分类，请你选一个：

候选与命中证据：
- **病案**（得分 X）：标题命中 [..]；正文命中 [..]
- **护理**（得分 Y）：标题命中 [..]；正文命中 [..]
- **健康档案**（得分 Z）：标题命中 [..]；正文命中 [..]

回复 `病案` / `护理` / `健康档案` 之一即可，或回复 `跳过` 终止本次归档。
```

无任何命中时模板：

```markdown
该文档关键词未命中三类业务域词表（病案 / 护理 / 健康档案），请你直接指定分类，或回复 `跳过` 终止本次归档。
```

## 不要做的事

- 不要在三类之外凭直觉新增分类
- 不要因为一两词稍占优势就强行选边
- 不要因用户上一份文档归到某类就默认本次同类
